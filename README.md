# ConceptInfra - Mashriq Heights

Premium residential project website for **Mashriq Heights** by ConceptInfra, located in Aligarh, Uttar Pradesh, India.

## Project Overview

Mashriq Heights is a thoughtfully crafted residential project developed by ConceptInfra, a professionally driven real estate and construction development firm. This website serves as the primary digital presence for the project, featuring property details, amenities, location advantages, and enquiry forms.

## Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Custom properties, Grid, Flexbox, animations
- **Vanilla JavaScript** - No frameworks or dependencies
- **Google Fonts** - Playfair Display, Montserrat, Cormorant Garamond

## File Structure

```
conceptinfra/
├── index.html    # Main HTML file with all sections
├── styles.css    # Complete stylesheet with responsive design
├── script.js     # Interactive features and animations
└── README.md     # This file
```

## Website Sections

1. **Hero** - Full-screen landing with animated tagline and CTA buttons
2. **Walkthrough Video** - Video placeholder section for project walkthrough
3. **About Us** - Company background, legacy, and quality promise
4. **Leadership** - Founder (Dr. M Y Khan) and Principal Developer (Shariq Khan)
5. **Expert Consultants** - Architect, Structure Consultant, Interior & Exterior Designer
6. **Why Concept Infra** - Six key value propositions
7. **Vision & Mission** - Company mission and vision statements
8. **Architectural Concept** - 4BHK Suites, Community Hall, Amenities, Natural Elements, Recreation
9. **World-Class Amenities** - 19 amenities including gated society, mosque, sky lounge, gymnasium, etc.
10. **Brochure / Site Plans** - Download form (name, email, phone)
11. **Location Advantages** - 6 proximity highlights with embedded Google Map
12. **Enquire Now** - Contact form with interest selection
13. **Footer** - Company info, project links, social media, contact details

## Features

- Responsive design (desktop, tablet, mobile)
- CSS-only preloader animation
- Scroll-triggered reveal animations (IntersectionObserver)
- Animated stat counters
- Typewriter effect on hero tagline
- Parallax hero background
- Sticky header with scroll state
- Mobile hamburger navigation
- Active nav link tracking on scroll
- Smooth scroll to sections
- Floating "Enquire Now" CTA button
- Back-to-top button
- Form validation with notification system
